คุณคือผู้เชี่ยวชาญด้าน Front-end Developer และ Accessibility Web Game 
จงนำซอร์สโค้ดไฟล์ HTML เกมเสียง (Audio Game) ที่ฉันจะแนบต่อไปนี้ ไปปรับปรุงเฉพาะ "เลเยอร์การมองเห็นและกราฟิก (CSS & HTML Decoration)" ตามเงื่อนไขที่กำหนดอย่างเคร่งครัด

[เงื่อนไขสำคัญที่สุด - ห้ามละเมิดเด็ดขาด]
1. ห้ามแก้ไข เปลี่ยนแปลง หรือลบโค้ดในส่วน JavaScript Logic, ระบบการคำนวณคะแนน, ตัวแปรสถานะเกม, ระบบตรวจจับ Keydown/Movement DeviceMotion และระบบเล่นเสียง AudioContext เดิมที่มีอยู่แล้ว 100%
2. **การซ่อนจาก Screen Reader (SC)**: องค์ประกอบกราฟิกและสิ่งตกแต่งที่เพิ่มเข้ามาใหม่ทั้งหมด (เช่น ต้นไม้, ท้องฟ้า, เมฆ, วัตถุข้างทาง) จะต้องถูกซ่อนจาก Screen Reader อย่างสมบูรณ์ โดยต้องใส่ attribute `aria-hidden="true"` พร้อมตั้งค่า CSS `pointer-events: none; user-select: none;` เพื่อไม่ให้ระบบอ่านหน้าจอตรวจจับหรือรบกวนผู้เล่นทางสายตา

[รายการสิ่งที่ต้องปรับปรุงด้าน Visual]
1. หัวของตัวละคร (Character Head): ตอนนี้มุมมองเห็นด้านหลังของหัวถูกต้องแล้ว แต่ให้เอาเส้นขีดขวางตรงท้ายทอยออก (ลบ Style หรือ Selector ที่สร้างเส้นขีดตรงหัวออก) ให้หัวด้านหลังดูเคลียร์และเรียบเนียน
2. ลำตัว แขน ขา และหาง (Back-View Running Perspective): ปรับแอนิเมชันและการจัดเลเยอร์ (z-index) ของส่วนใต้หัวลงมาทั้งหมด ให้สัมพันธ์กับการวิ่ง "หันหลังมุ่งหน้าเข้าหาถนน" (Forward-Run) โดยเวลาแขนหรือขาก้าวไปข้างหน้า (ห่างจากผู้เล่น) ให้ใช้ CSS keyframes ปรับขนาดให้เล็กลงเล็กน้อยและหลบไปอยู่หลังลำตัว ส่วนหางให้อยู่เลเยอร์ด้านหลังตัวอย่างสมบูรณ์แบบ เพื่อไม่ให้ดูขัดแย้งกับหัวด้านหลัง
3. ท้องฟ้า (Vibrant Blue Sky): ปรับเปลี่ยนพื้นหลังเลเยอร์ท้องฟ้า จากเดิมที่เป็นสีดำ ให้เป็นสีฟ้าสดใส (Bright Blue Sky) ไล่เฉดสีให้ดูเคลียร์ตาและสว่างสดชื่น
4. ฉากข้างทางสลับซ้ายขวา (Asymmetrical Roadside Objects): เพิ่มแอนิเมชันวัตถุข้างถนนฝั่งซ้ายและขวา (เช่น ต้นไม้สูง-เตี้ย, ตู้ไปรษณีย์, ป้ายจราจร) ให้วิ่งสวนเลนเยื้องสลับฝั่งกันอย่างอิสระแบบไม่สมมาตร (ไม่โผล่มาพร้อมกันเป็นคู่) เพื่อจำลองบรรยากาศการอยู่บนท้องถนนจริง ๆ

กรุณาส่งโค้ด HTML ฉบับสมบูรณ์ที่ผ่านการแก้ไขตามคำสั่งด้านบนกลับมาให้ฉัน

*** สำคัญที่สุด ปรับเฉพาะสิ่งที่ร้องขอ ส่วนอื่นๆที่ไม่เกี่ยวข้องต้องเหมือนเดิมแบบ 100%
Code ห้ามหาย ต้องมาครบ ปรับได้เฉพาะสิ่งที่สรุปตามนี้เท่านั้น รวมทั้งหมดมาเลย
