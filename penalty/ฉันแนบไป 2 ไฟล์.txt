ฉันแนบไป 2 ไฟล์
เริ่มที่ d.html ให้คุณศึกษาการเล่นไฟล์เสียงของไฟล์นี้ ซึ่งเป็นการทำงานทุกไฟล์เสียงที่ฉันชอบมากๆ
และการสั่งการจาก firebase เกี่ยวกับเสียงด้วย ศึกษาให้พร้อม รวมถึงถ้ามีการจัดคิวเล่นไฟล์เสียง
ให้นำมาปรับใช้กับไฟล์ o.html ให้ทำงานแบบเดียวกันเป๊ะเลยได้ไหม
แต่ไฟล์เสียงที่ใช้กับ o.html จะมีเพิ่มเติมเพราะเพิ่มการสุ่มเสียงมาเพิ่ม
การสุ่มนี้ต้องสุ่มแบบไม่ซ้ำกันเลย และเล่นต่อเนื่องได้แบบวนซ้ำไปเรื่อยได้ไม่มีวันหมด
แต่ทั้งหมดอยากปรับให้การทำงานของ audio เหมือนกันกับไฟล์ d.html
โดยที่ต้องไม่ไปแตะกับการเล่น การคำนวณการแพ้ชนะ และโครงสร้างที่ดีทั้งหมดอยู่แล้ว
ย้ำไม่ลด code ตัด code ของฉันที่ไม่ได้ขอร้องนะ
รบกวนปรับและรวม code ทั้งหมดมาให้ด้วยนะครับ

ฉันแนบไฟล์ 2 ไฟล์
o.html
รบกวนเช็คเรื่องกฎกติกาจากไฟล์ o.html ไฟล์ล่าสุดนี้
เอาการคำนวณการแพ้ชนะจากไฟล์ o.html มาปรับใช้ในไฟล์ a.html
เพราะกติกาของ o.html ถูกต้อง โดยให้ปรับแค่ส่วนนี้เท่านั้นใน a.html
